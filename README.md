# wan22-animate-Boris-autoinstall

Auto-installer for Wan 2.2 Animate models in ComfyUI (Boris Edition).

This script downloads all required models for:
- Wan 2.2 Animate (main model)
- Character swap
- Lip-sync
- Relight LoRA
- Detection models (YOLOv10 + ViTPose)
- Clip Vision
- Text Encoder
- VAE (shared with WanVideo)

Fully compatible with Smushnikov’s ComfyUI template.
